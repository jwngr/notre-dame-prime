# Notre Dame Prime

A prime number in the shape of the Notre Dame monogram.

## Notes

Run site:

```bash
# From website/ directory
$ npm run dev
```

Run flask locally:

```bash
# From server/ directory
$ export FLASK_APP=server.py FLASK_DEBUG=1 FLASK_RUN_PORT=3373
$ flask run
```
